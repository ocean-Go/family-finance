## ✨ 功能特点

### 🏠 首页
- 本月收支概览
- 快速统计卡片
- 最近交易记录

### ➕ 智能添加
- 📷 图片上传（小票/收据/账单）
- ✏️ 文字描述输入
- 🤖 AI 智能识别
  - 自动提取金额
  - 自动识别商家
  - 自动判断收支类型
  - 自动建议分类

### 📋 交易记录
- 完整交易列表
- 按日期/分类筛选
- 金额明细查看

### 📊 数据分析
- 分类支出占比
- 月度趋势对比
- 💡 智能建议提醒

## 🛠️ 技术栈

- **前端**: 原生 HTML5 + CSS3 + ES6+
- **AI**: 关键词智能分析

- **部署**: GitHub Pages
- **PWA**: 支持添加到主屏幕

## 🚀 快速开始

### 本地运行

```bash
cd family-finance
python3 -m http.server 3000
# 浏览器打开 http://localhost:3000
```

### 部署到 GitHub

```bash
# 1. 创建 GitHub 仓库
gh repo create family-finance --public

# 2. 推送代码
git add .
git commit -m "✨ Initial commit"
git push

# 3. 启用 GitHub Pages
# Settings → Pages → Source: main branch → Save
```
